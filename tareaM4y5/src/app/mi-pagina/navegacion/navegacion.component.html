Enrutamiento y Navegacion

Prerrequisitos

Antes de crear una ruta, debe estar familiarizado con lo siguiente:

  • Conceptos básicos de componentes
  • Conceptos básicos de las plantillas
  • Una aplicación angular: puede generar una aplicación angular básica utilizando la CLI de Angular

Genere una aplicación con el enrutamiento habilitado

El siguiente comando usa la CLI de Angular para generar una aplicación Angular básica con un módulo de enrutamiento de aplicaciones, llamado AppRoutingModule, que es un NgModule donde puede configurar sus rutas. El nombre de la aplicación en el siguiente ejemplo es routing-app.

ng new routing-app --routing

Agregar componentes para enrutamiento

Para usar el enrutador angular, una aplicación debe tener al menos dos componentes para que pueda navegar de uno a otro.

Importando sus nuevos componentes

Para usar sus nuevos componentes, impórtelos AppRoutingModuleen la parte superior del archivo, de la siguiente manera:

import { FirstComponent } from './first/first.component';

import { SecondComponent } from './second/second.component';

Definiendo una ruta básica

Hay tres bloques de construcción fundamentales para crear una ruta.

Importe el AppRoutingModule en el archivo AppModule y agréguelo a la matriz imports.

import { BrowserModule } from '@angular/platform-browser';

import { NgModule } from '@angular/core';

import { AppRoutingModule } from './app-routing.module';

import { AppComponent } from './app.component';

@NgModule({

  declarations: [

    AppComponent

  ],

  imports: [

    BrowserModule,

    AppRoutingModule

  ],

    providers: [],

    bootstrap: [AppComponent]

  })

  export class AppModule { }

1. Importe RouterModuley Routesa su módulo de enrutamiento.

La CLI de Angular realiza este paso automáticamente. La CLI también configura una Routesmatriz para sus rutas y configura las matrices importsy exportspara .@NgModule()

import { NgModule } from '@angular/core';

import { Routes, RouterModule } from '@angular/router'; // CLI imports router

const routes: Routes = []; // sets up routes constant where you define your routes

@NgModule({

imports: [RouterModule.forRoot(routes)],

exports: [RouterModule]

})

  export class AppRoutingModule { }

2. Defina sus rutas en su matriz Routes.

Cada ruta de esta matriz es un objeto JavaScript que contiene dos propiedades. La primera propiedad, pathdefine la ruta URL de la ruta. La segunda propiedad componentdefine el componente que Angular debe usar para la ruta correspondiente.

const routes: Routes = [

  { path: 'first-component', component: FirstComponent },

  { path: 'second-component', component: SecondComponent },

];

3. Agrega tus rutas a tu aplicación.

Ahora que ha definido sus rutas, puede agregarlas a su aplicación. Primero, agregue enlaces a los dos componentes. Asigne la etiqueta de anclaje a la que desea agregar la ruta al routerLinkatributo. Establezca el valor del atributo en el componente para que se muestre cuando un usuario hace clic en cada enlace. A continuación, actualice la plantilla de su componente para incluir . Este elemento informa a Angular que actualice la vista de la aplicación con el componente para la ruta seleccionada.<router-outlet>

          <li><a routerLink="/second-component" routerLinkActive="active">Second Component</a></li>

      </ul>

  </nav>
