Data Binding

La vinculación de datos es el mecanismo que vincula la interfaz de usuario o la interfaz de usuario de las aplicaciones a los modelos. Mediante Data Binding, el usuario podrá manipular los elementos presentes en el sitio web utilizando el navegador. Por lo tanto, siempre que se haya cambiado alguna variable, ese cambio en particular debe reflejarse en el Modelo de objetos de documento o DOM .

En Angular , Data Binding define la interacción entre los componentes y el DOM. El enlace de datos es parte de todas las versiones de Angular, desde AngularJS hasta la última versión de Angular 9 .

Tipos de enlace de datos en angular

Angular permite el enlace de datos unidireccional y bidireccional. El enlace de datos unidireccional es un tipo simple de enlace de datos en el que se le permite manipular las vistas a través de los modelos. Esto implica que realizar cambios en el código de TypeScript se verá reflejado en el HTML correspondiente . En Angular, el enlace de datos unidireccional se logra mediante:

  • Interpolación o interpolación de cadenas
  • Vinculación de propiedad
  • Enlace de eventos

El enlace de datos bidireccional, por otro lado, permite la sincronización de datos de tal manera que las vistas se pueden actualizar usando los modelos y los modelos se pueden actualizar usando vistas. Esto significa que su aplicación podrá compartir información entre una clase de componente y su plantilla.

Enlace de datos unidireccional

En la vinculación de datos unidireccional, los datos fluyen solo en una dirección, es decir, de los modelos a las vistas. Como se mencionó anteriormente, el enlace de datos unidireccional en Angular puede ser de tres tipos, es decir, interpolación, enlace de propiedad y enlace de eventos.

Enlace de interpolación

El enlace de interpolación se utiliza para devolver la salida HTML del código TypeScript, es decir, de los componentes a las vistas. Aquí, la expresión de la plantilla se especifica entre llaves dobles. Mediante la interpolación, se pueden agregar cadenas al texto que está presente entre las etiquetas de elementos HTML y dentro de las asignaciones de atributos. Estas cadenas se calculan mediante expresiones de plantilla.

EJEMPLO:

<h1> { { title } } </h1>

Learn <b> { { course } }

</b> with Edureka.

2 * 2 = {{2 * 2}}

<div><img src="{ { image } }"></div>

La parte de TypeScript de este código es la siguiente:

export class AppComponent {

title = 'Databinding';

course ='Angular';

image = 'paste the url here'

}
